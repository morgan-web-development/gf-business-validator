== Description ==
A plugin for Gravity Forms that allows you to restrict specific email domains like Gmail and Outlook for email fields.

== Installation ==
1. Upload the `gravity-email-domain-restrictor` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the checkbox option in Gravity Forms email fields to enable domain restrictions.
